# RealTimeTranslator (基于大模型的 Galgame 实时翻译器)

这是一个基于本地系统剪贴板和深层引擎 Hook 的 Ren'Py (Galgame) 实时翻译系统。
配合大模型 (如 DeepSeek) API，能够实现极低延迟、极高准确度的上下文连贯翻译。

## 核心特性
- **底层 Hook 技术**：抛弃传统视觉截屏 OCR，直接注入 Ren'Py 游戏引擎，在文字渲染前瞬间截获拼接好的全句。完美解决续写 (extend)、NVL模式、以及魔改 UI 导致的文字残缺问题。
- **强制抓取模式**：支持 `T` 键作为强制雷达，直接扫描屏幕物理组件。对于屏蔽了底层变量的重度魔改游戏，依然能强制获取当前屏幕所有文字。
- **极低 Token 消耗的上下文机制**：带有“滑动窗口记忆机制”，自动将上一句的翻译结果作为下一句的上下文提示词（Prompt）传给大模型，让翻译具备极强的语境连贯性和代词（他/她/它）准确度。
- **标点符号智能过滤**：自动忽略纯标点符号的翻译请求，防止大模型乱回复。

## 文件结构
- `main.py`:  PyQt6 构建的无边框、置顶浮动透明字幕窗口。
- `translator_engine.py`: 后台翻译引擎，负责调用大模型 API 和本地缓存管理。
- `requirements.txt`: Python 依赖项。
- `renpy_plugin/00antigravity_hack.rpy`: 游戏端截取脚本（扔进游戏的 `game/` 目录下即可生效）。

## 部署与使用
1. **安装环境**
   ```bash
   pip install -r requirements.txt
   ```
2. **配置 API 密钥**
   - 复制 `.env.example` 为 `.env`
   - 填入你的大模型 API 密钥 (API_KEY) 和 Base URL。切记**不要**将包含你真实 API Key 的 `.env` 文件上传到公开仓库！
3. **注入游戏**
   - 将 `renpy_plugin/00antigravity_hack.rpy` 文件复制到你要玩的 Ren'Py 游戏的 `game/` 文件夹内。
4. **启动翻译器**
   ```bash
   python main.py
   ```
5. **在游戏中操作**
   - 游戏默认开启“全自动翻译”模式。每次点击鼠标，下一句话就会自动在顶层字幕框中被翻译。
   - 按下 `Y` 键：切换自动/手动模式。
   - 按下 `T` 键：手动强制刷新并抓取当前屏幕上的文字进行翻译（用于应对极端魔改游戏）。

## 注意事项
请确保不要在公共仓库中暴露你的 `.env` 文件，该文件已被加入 `.gitignore` 保护。
